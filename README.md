# 🦦 Otter Gift Hunt – Birthday Mini-Game

Otter Gift Hunt is a small, cute, browser-based birthday mini-game created as a fun and interactive surprise.  
Players help a cheerful otter collect floating gift boxes 🎁 while avoiding bubbles 🫧. Reaching the goal unlocks a special birthday message written with love from a sibling.

The game is lightweight, mobile-friendly, and requires no installation — it runs entirely in the browser and is shared via a simple link.

---

## 🎮 How to Play
1. Click **Start**
2. Tap / click the floating **🎁 gifts** to gain points
3. Avoid the **🫧 bubbles** (they reduce your score)
4. Reach the goal before time runs out to unlock the birthday message 🎂

---

## ✨ Features
- 🦦 Cute otter-themed design
- 🎁 Simple tap / click gameplay
- 🎉 Birthday message revealed upon winning
- 🔗 Shareable via URL
- 📱 Works on both mobile and desktop
- ⚡ Optimized for smooth performance
